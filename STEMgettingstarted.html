Tips on Getting Started with STEM Contests

Home Updates Tips


Scholarstein has created a planning worksheet to help you brainstorm ideas for STEM contests.

Firstly, think about your current computing skills. These could include being able to use Scratch block code.


Then, write down any global or local problems you observe in your everyday life. These could be global warming or a simple improvement that you see can be added to your local library.


Next, comes the part that could take a while. Try linking the skills you have to the problems that need to be solved. The link should answer the question, how can I use this skill to solve this problem feasibly and practically?
